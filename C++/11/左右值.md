**this**

在类的非静态成员类型中，编译器会隐式的为函数添加一个参数：

``` ClassName* const this```

**不能对this取地址**，也不能reassign，所以是个右值。

但是*this的类型是ClassName&，所以是个左值。

C++ 11以后可以通过右值引用限定函数

```
class X {
public:
    void f() &  { std::cout << "lvalue this\n"; }
    void f() && { std::cout << "rvalue this\n"; }
};
```

this返回

```
X* const
X* const&&
```

但在这里this仍然是右值。**this不占用sizeof()。**
