### 作业8

几何中曲线和曲面。

读入相关点的数据，采用分段的方法，每四个点一组(这里最后要有边界处理)按照公式计算即可。

```
if(num == 2){
    res = (1 - t) * p[0] + t * p[1];
}else if(num == 3){
    res = (1 - t) * (1 - t) * p[0] + 2 * (1 - t) * t * p[1] +
    t * t * p[2];
}else{
    res = (1 - t) * (1 - t) * (1 - t) * p[0] +
    3 * (1 - t) * (1 - t) * t * p[1] + 3 * t * t * (1 - t) * p[2]+
    t * t * t * p[3];
}
```

```
float b0 = (1 - t) * (1 - t) * (1 - t) / 6,
b1 = (3 * t * t * t - 6 * t * t + 4) / 6,
b2 = (-3 * t * t * t + 3 * t * t + 3 * t + 1) / 6,
b3 = t * t * t / 6;
auto p = b0 * vertices[index - 3] + b1 * vertices[index - 2] +
 	b2 * vertices[index - 1] + b3 * vertices[index];
return p;
```

曲面的话按照细分的粒度，相当与生成多条曲线，将曲线连接即可。

```
vector<Vertex>       vertices;
vector<unsigned int> indices;
vector<Texture>      textures;
unsigned int VAO;
```

mesh类中主要有两个函数一个是初始化。其中包括设置VAO、VBO、EBO和加载纹理等内容。

```
// 顶点位置
glEnableVertexAttribArray(0);   
glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, sizeof(Vertex), (void*)0);
// 顶点法线
glEnableVertexAttribArray(1);   
glVertexAttribPointer(1, 3, GL_FLOAT, GL_FALSE, sizeof(Vertex), (void*)offsetof(Vertex, Normal));
// 顶点纹理坐标
glEnableVertexAttribArray(2);   
glVertexAttribPointer(2, 2, GL_FLOAT, GL_FALSE, sizeof(Vertex), (void*)offsetof(Vertex, TexCoords));
```

另一个函数就是渲染函数，将纹理单元加载到片段着色器的纹理采样器中。
