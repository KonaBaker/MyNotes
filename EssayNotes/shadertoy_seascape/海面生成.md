# 海洋高度生成的完整数学流程

## 概览：从随机数到真实海浪

```
随机种子 → 伪随机场 → Perlin噪声 → 域扭曲 → 波形塑造 → 分形叠加 → 最终海面
   ↓          ↓         ↓        ↓        ↓        ↓        ↓
 hash()   →  noise()  → sea_octave() → map() → 海面高度
```

---

## 第一层：hash() - 伪随机数基础

```glsl
float hash( vec2 p ) {
    float h = dot(p,vec2(127.1,311.7));    
    return fract(sin(h)*43758.5453123);
}
```

### 数学原理
**目标**：将2D整数坐标映射为[0,1]范围内的"随机"数

**算法步骤**：



1. **线性组合**：`dot(p, vec2(127.1, 311.7))`
   - 将(x,y)映射为标量：h = 127.1×x + 311.7×y
   - 127.1和311.7是精心选择的素数，避免周期性

2. **非线性变换**：`sin(h)`
   - 利用正弦函数的混沌特性
   - 即使输入很接近，输出也可能差别很大

3. **放大和混合**：`sin(h) × 43758.5453123`
   - 43758.5453123是魔法数字，最大化随机性
   - 来源于数论研究，具有良好的统计特性

4. **取小数部分**：`fract()`
   - 确保输出在[0,1]范围内
   - 创建伪随机分布

### 视觉化理解
```
输入网格点:
(0,0) → hash → 0.374
(1,0) → hash → 0.891  
(0,1) → hash → 0.156
(1,1) → hash → 0.643

看起来随机，但实际上是确定性的！
```

**关键特性**：
- **确定性**：相同输入总是产生相同输出
- **均匀分布**：输出在[0,1]均匀分布
- **低相关性**：相邻点的输出看似无关

## 第二层：noise() - Perlin噪声场

```glsl
float noise( in vec2 p ) {
    vec2 i = floor( p );        // 网格坐标
    vec2 f = fract( p );        // 网格内位置
    vec2 u = f*f*(3.0-2.0*f);   // 平滑插值函数
    
    return -1.0+2.0*mix( 
        mix( hash( i + vec2(0.0,0.0) ), hash( i + vec2(1.0,0.0) ), u.x),
        mix( hash( i + vec2(0.0,1.0) ), hash( i + vec2(1.0,1.0) ), u.x), 
        u.y);
}
```

### 数学构造过程

#### 1. 空间离散化
```glsl
vec2 i = floor(p);  // 例如：p=(2.3, 1.7) → i=(2, 1)
vec2 f = fract(p);  // f=(0.3, 0.7)
```

将连续空间分割为单位正方形网格。

#### 2. 平滑插值函数
```glsl
vec2 u = f*f*(3.0-2.0*f);  // 三次Hermite插值
```

**数学形式**：u(t) = t²(3-2t) = 3t² - 2t³

**特性分析**：
```
u(0) = 0, u(1) = 1      // 边界条件
u'(0) = 0, u'(1) = 0    // 导数连续性
u''(0) = 0, u''(1) = 0  // 二阶导数连续性
```

这确保了C²连续性，避免了明显的网格伪影。

#### 3. 双线性插值
```
网格单元：
hash(i+00) ——— hash(i+10)
    |             |
    |      P      |  
    |             |
hash(i+01) ——— hash(i+11)
```

**插值过程**：
1. 水平插值：bottom = mix(hash(i+00), hash(i+10), u.x)
2. 水平插值：top = mix(hash(i+01), hash(i+11), u.x)  
3. 垂直插值：result = mix(bottom, top, u.y)

#### 4. 值域重映射
```glsl
return -1.0+2.0*mix(...);  // [0,1] → [-1,1]
```

**目的**：增强对比度，让噪声有正负变化。

### 噪声特性
- **连续性**：C²连续，无明显跳跃
- **统计性**：近似正态分布，均值≈0
- **相关性**：相近点的值相关，远离点逐渐无关
- **各向同性**：在所有方向上统计特性相同

## 第三层：sea_octave() - 波浪形状塑造

```glsl
float sea_octave(vec2 uv, float choppy) {
    uv += noise(uv);                    // 域扭曲
    vec2 wv = 1.0-abs(sin(uv));        // 基础波形
    vec2 swv = abs(cos(uv));           // 辅助波形
    wv = mix(wv,swv,wv);               // 非线性混合
    return pow(1.0-pow(wv.x * wv.y,0.65),choppy);
}
```

### 第1步：域扭曲 (Domain Warping)
```glsl
uv += noise(uv);
```

**革命性创新**：不是在值域添加噪声，而是扭曲坐标空间！

**数学对比**：
```
传统方法: f(x,y) + noise(x,y)     // 值域叠加
域扭曲:   f(x+noise(x,y), y+noise(x,y))  // 坐标扭曲
```

**效果分析**：
```
原始坐标网格：     扭曲后的网格：
│ │ │ │ │          ╱ ╲ ╱ ╲ ╱
─ ─ ─ ─ ─    →    ╱   ╲   ╲   
│ │ │ │ │          ╲ ╱ ╲ ╱ ╲
─ ─ ─ ─ ─          ╱   ╲   ╲
│ │ │ │ │          ╲ ╱ ╲ ╱ ╲
```

**物理意义**：模拟流体的非线性变形，创造更自然的不规则性。

### 第2步：波形生成
```glsl
vec2 wv = 1.0-abs(sin(uv));   // 创建尖峰波
vec2 swv = abs(cos(uv));      // 创建圆润波
```

**波形分析**：

这两个函数画一下图就可以看出来，一个尖朝上，一个圆顶朝上。

### 第3步：混合
```glsl
wv = mix(wv,swv,wv);  // 使用wv作为混合权重！
```

**创新点**：混合权重不是常数，而是wv本身！

**数学展开**：

```
result = wv * (1-wv) + swv * wv = wv * (1-wv+swv)
```

**效果**：
- 当wv接近0（波谷）：主要使用wv（尖锐特征）
- 当wv接近1（波峰）：更多使用swv（圆润特征）
- 创造了**非均匀的波形变化**

### 第4步：最终塑形
```glsl
return pow(1.0-pow(wv.x * wv.y,0.65),choppy);
```

**数学解构**：
1. `wv.x * wv.y`：2D波形的乘积，创建交叉波纹
2. `pow(..., 0.65)`：压缩动态范围，柔化过渡
3. `1.0 - ...`：反转，让峰值变成谷值
4. `pow(..., choppy)`：根据风力调整尖锐度

**choppy参数的物理意义**：
```
choppy = 1.0: 温和海面，圆润波浪
choppy = 2.0: 中等海况，普通波浪  
choppy = 4.0: 恶劣天气，尖锐白帽浪
```

## 第四层：map() - 分形海洋构建

```glsl
float map(vec3 p) {
    float freq = SEA_FREQ;      // 基础频率 0.16
    float amp = SEA_HEIGHT;     // 基础振幅 0.6
    float choppy = SEA_CHOPPY;  // 基础尖锐度 4.0
    vec2 uv = p.xz; uv.x *= 0.75; // 各向异性
    
    float d, h = 0.0;    
    for(int i = 0; i < ITER_GEOMETRY; i++) {        
        d = sea_octave((uv+SEA_TIME)*freq,choppy);  // 正向波
        d += sea_octave((uv-SEA_TIME)*freq,choppy); // 反向波
        h += d * amp;        
        uv *= octave_m;    // 旋转缩放
        freq *= 1.9;       // 频率递增
        amp *= 0.22;       // 振幅递减
        choppy = mix(choppy,1.0,0.2); // 高频波更温和
    }
    return p.y - h;
}
```

### 分形理论基础

#### 1. 自相似性
**核心思想**：海洋在不同尺度上具有相似的统计特性

```
大波浪 (几十米): 基础海况
中波浪 (几米):   风生波
小波浪 (几十厘米): 毛细波
微波纹 (几厘米):  表面张力波
```

每一层都是前一层的"缩小+细化"版本。

#### 2. 功率谱理论
**Kolmogorov湍流定律**：E(k) ∝ k^(-5/3)

在这个shader中：
```
频率增长: freq *= 1.9  
振幅衰减: amp *= 0.22
功率衰减: (0.22)² ≈ 0.048
指数: log(0.048)/log(1.9) ≈ -5.0
```

虽然不完全匹配理论值-5/3，但在视觉上非常接近真实海洋。

### 关键算法细节

#### 1. 双向波浪传播
```glsl
d = sea_octave((uv+SEA_TIME)*freq,choppy);  // 正向传播
d += sea_octave((uv-SEA_TIME)*freq,choppy); // 反向传播
```

**物理意义**：
- 真实海洋中波浪从多个方向传播
- 正向波：主导风向产生的波浪
- 反向波：反射波、涌浪等

**数学效果**：
```
f(x-ct) + f(x+ct)  // 标准的波动方程解
```

#### 2. Octave变换矩阵
```glsl
const mat2 octave_m = mat2(1.6,1.2,-1.2,1.6);
uv *= octave_m;
```

**矩阵分析**：
```
|1.6  1.2|  = 旋转 × 缩放
|-1.2 1.6|
```

- **旋转角度**：θ = arctan(1.2/1.6) ≈ 36.87°
- **缩放因子**：√(1.6² + 1.2²) = 2.0
- **作用**：避免不同octave的网格对齐，增加随机性

#### 3. 各向异性处理
```glsl
uv.x *= 0.75;  // X方向压缩
```

**物理动机**：
- 模拟主导风向的影响
- 真实海洋很少完全各向同性
- 创造更自然的波浪方向性

#### 4. 自适应choppy
```glsl
choppy = mix(choppy,1.0,0.2);  // 向1.0收敛
```

**设计思想**：
- 大尺度波浪：可以很尖锐（受风力影响）
- 小尺度波浪：趋向平缓（被表面张力平滑）
- 符合流体力学的多尺度特性

## 第五层：时间动态和最终计算

### 时间演化
```glsl
#define SEA_TIME (1.0 + iTime * SEA_SPEED)
```

**波浪传播**：
```
波速公式: c = √(gλ/(2π))  // 深水波
其中 g=9.8m/s², λ是波长
```

不同频率的波浪有不同的传播速度，高频波传播较慢。

### 最终高度计算

给定空间点 P(x, y, z)，海面高度的确定过程：

#### 第1步：提取水平坐标
```glsl
vec2 uv = p.xz;  // 忽略y坐标，只看水平位置
```

#### 第2步：多层分形叠加
```glsl
float h = 0.0;
for(每个octave) {
    h += octave_contribution;
}
```

#### 第3步：高度场函数
```glsl
return p.y - h;  // 距离场函数
```

**函数意义**：
- 返回值 > 0：点在海面上方（空气中）
- 返回值 = 0：点在海面上（表面）
- 返回值 < 0：点在海面下方（水中）

## 完整示例：计算一个具体点的海面高度

### 给定条件
```
查询点: P = (10.0, ?, 20.0)  // 我们要找对应的海面高度
时间: t = 5.0秒
```

### 计算流程

#### 第1步：初始参数
```glsl
float freq = 0.16;
float amp = 0.6;  
float choppy = 4.0;
vec2 uv = vec2(10.0, 20.0);
uv.x *= 0.75;  // uv = (7.5, 20.0)
float h = 0.0;
```

#### 第2步：第一层octave (i=0)
```glsl
// 计算正向波
vec2 uv_forward = (uv + SEA_TIME) * freq = (7.5+5.8, 20.0+5.8) * 0.16
d1 = sea_octave(uv_forward, 4.0)

// 计算反向波  
vec2 uv_backward = (uv - SEA_TIME) * freq = (7.5-5.8, 20.0-5.8) * 0.16
d2 = sea_octave(uv_backward, 4.0)

// 叠加
h += (d1 + d2) * 0.6
```

#### 第3步：变换参数
```glsl
uv *= octave_m;     // 旋转缩放
freq *= 1.9;        // freq = 0.304
amp *= 0.22;        // amp = 0.132  
choppy = mix(4.0, 1.0, 0.2) = 3.4
```

#### 第4步：第二层octave (i=1)
重复上述过程...

#### 第5步：第三层octave (i=2)
重复上述过程...

#### 第6步：最终结果
```glsl
// 假设经过3次迭代后，h = 0.734
// 对于查询点 P(10.0, y, 20.0)
// 海面高度为 y = h = 0.734
// 距离场函数: f(10.0, y, 20.0) = y - 0.734
```

**海面位置确定**：当 f(P) = 0 时，P就在海面上，此时 y = 0.734米。
