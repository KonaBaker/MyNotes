## Fog 雾效

### depth fog

- linear fog

  ```
  factor = (end - z)/(end - start);
  ```

- exp fog

  ```
  factor = exp(- density * z);
  ```

- exp squared fog

  ```
  factor = exp(-(density * z)^2);
  ```
