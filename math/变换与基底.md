**行主序和列主序**

对于矩阵：

```c++
M = [a  b  c  d]
    [e  f  g  h]
    [i  j  k  l]
    [m  n  o  p]
```

对于列主序，是先存储列 

```c++
struct Matrix4 {
    Vector4 col0;  // [a, e, i, m]
    Vector4 col1;  // [b, f, j, n]
    Vector4 col2;  // [c, g, k, o]
    Vector4 col3;  // [d, h, l, p]
};
```

```c++
mat[0] == [a, e, i, m]
mat[0][1]== e
```

在变换的时候列主序的矩阵要在左边，列向量在右边

旋转轴：ux, uy, uz（正交单位向量）

平移：T = [Tx, Ty, Tz]

列主序需要如下构造：

```c++
Matrix4 mat(
    Vector4(ux.x, ux.y, ux.z, 0),  // 第1列：旋转矩阵的第1列
    Vector4(uy.x, uy.y, uy.z, 0),  // 第2列：旋转矩阵的第2列
    Vector4(uz.x, uz.y, uz.z, 0),  // 第3列：旋转矩阵的第3列
    Vector4(Tx,   Ty,   Tz,   1)   // 第4列：平移向量
);
```

数学形式是这样的：

```c++
[ux.x  uy.x  uz.x  Tx]
[ux.y  uy.y  uz.y  Ty]
[ux.z  uy.z  uz.z  Tz]
[  0     0     0    1]
```



**列主序和行主序代码表示方式以及存储方式不同，在计算上有差异，数学形式也是不一样的**
