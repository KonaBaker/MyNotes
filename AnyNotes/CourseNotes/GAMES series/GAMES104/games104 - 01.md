## 2. Deep In

---

### 2.1 Resource Layer

管理资产的整个生命周期

-  **资源 --> 资产**

​	数据引擎化，将各式各样的资源文件，各种各样的格式，转化成引擎的资产，这种资产可以在GPU中高效的运行。

- **关联文件**

​	composite asset file 指明资产之间的关联的文件，一个机器人的纹理，颜色，顶点的不同文件的联系。GUID 资产识别号

- **handle系统**
- **内存管理**

​	根据玩家进度不断下载和释放资源。

### 2.2 Function Layer

tick 两块大的内容 每一个_" 普朗克时间"_ 所做的动作

| tick logic    | tick render     |
| ------------- | --------------- |
| 用于模拟\物理 | 用于渲染\生成图 |
| Camera        | RenderCamera    |
| Motor         | culling         |
| Controller    | rendering       |
| Animation     | postprocess     |
| Physics       | present         |
